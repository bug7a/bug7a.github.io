USER'S MANUAL
This mobile application; has been designed with simplicity, functionality, and fast usability in mind.

STATUS
  • Green and yellow colors in graphics; It shows that spending is less or more compared to last month.
  • The color purple indicates that there is no expense to be compared.
  • The boundary line and its value on the chart; shows last month's expenditures.


  • This month, $1768 of the $2500 budget has been spent.
  • Last month, these times; A total of $1110 was spent.
  • Since this month's expenses are higher than last month's; The graph appears in yellow.
  • This month, the remaining budget is $732.
  • This month, the category with the most spending; The "HOBBY" category became the category with 32% of the total expenditures.
  • Last month, spending in the "HOBBY" category was $745, while this month is $573, which is less than last month.
RECEIPTS
  • To add a new expense; You can use the "+" button at the bottom right.
  • When an expense is pressed, that expense can be edited again.
  • Expenses can be filtered by category name or description.
  • Old charges from inactive categories continue to appear in gray.
TOOLS / BUDGET
  • You can set this month's budget from the TOOLS page.
  • Determined budget; It can be changed repeatedly within a month.
  • The budgets for the past months cannot be rearranged.
TOOLS / CATEGORIES
  • The icons of active categories are colored. You can change the status of a category by pressing its icon.
  • You can edit the name of a category by pressing its name.
  • Categories with expenditures added in the last month cannot be disabled.
  • Old expenses incurred in inactive categories are included in the calculations. However, the visibility and editing of expenditures are restricted.
GENERAL INFORMATION
  • You can use this application free of charge for the first four months.
  • All data is stored within the device.
  • The application stores data for a maximum of 12 months retrospectively.
  • Old data is automatically cleared.
  • Works offline, no need for internet.
  • Currency and fractional value input is not used.
  • Added data cannot be backed up or moved.
